# 预览版本 V1.0.0 Alpha

:::danger
这是一个早期测试版本，不建议真的用来管理你的游戏存档
:::

**注意**
- 这是一个 alpha 测试版，请小心使用
- 本软件还不兼容老版本存档，请看 #38

**更新内容**
- 替换引擎为 Rust + Tauri
- 体积缩小到 15MB 以内
- 支持存档多文件、文件夹
- 安全性提升
- 修复启动其他文件不正常的问题
- 增加打开备份目录的功能
- 修复无法储存空文件夹的bug


**提醒**
- 本软件在征集图标，有兴趣请看 [#34](https://github.com/mcthesw/game-save-manager/discussions/34)
- 本项目正在讨论关于注释的问题，请看 [#33](https://github.com/mcthesw/game-save-manager/discussions/33)
- 本项目在寻求合作者，如果你也对此感兴趣，可以加入我们
