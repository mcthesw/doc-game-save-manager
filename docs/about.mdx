# 关于

## 关于游戏存档管理器
存档管理器本来只是在我自己玩法环的时候，为了方便管理存档而写的一个小工具，后来应室友要求加了一些功能，在法环发售初期，我们用它刷了很多卢恩，后来发现有人用bug来损坏他人存档，我就公开发布了这个软件

现在它已经逐渐演变成了一个开源项目，功能也逐渐完善，希望能帮助到更多的玩家，如果你愿意也出一份力，也可以通过 github 来参与开发，即使你不会编程，也可以通过提交 issue 来提出你的建议，或者在本网站帮忙撰写文档
