# 我安装到了错误的位置
卸载程序会将安装目录下的所有文件删除，如果你安装到了错误的位置，那么卸载程序是很危险的。好在这个问题很好解决，请按照以下步骤操作：

假设你在Windows系统下使用安装包安装了这个程序，安装目录是`H:\apps\`，而这里面装有很多你的其他软件，现在你想删掉存档管理器或重新安装到正确的地方（比如一个空目录），却想保留文件夹内其他文件

1. 找到卸载程序，它在在安装目录下，双击运行（也可以通过设置面板点击卸载）
2. 进入卸载面板，不要勾选图中的选项（只有勾选后会清空当前文件夹）
![卸载程序](/img/help/uninstall.jpg)
3. 点击卸载按钮，完成卸载

此时，卸载程序会删除该文件夹内的`RGSM.exe`和一些缓存文件，而其他文件会保留下来，如果你想完全清除存档管理器的数据，这里需要手动删除程序的配置和存档文件，它们分别是：

1. GameSaveManager.config.json
2. save_data

现在，存档管理器已经正确卸载
