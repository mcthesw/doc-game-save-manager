# 设置说明

## 当未描述存档时，弹出提示
指的是你在管理游戏存档界面，未在顶栏对应的输入框中输入描述信息时，是否弹出提示要求输入描述信息

## 当自动备份时，弹出提示
这个选项依赖于[退出到托盘](#退出到托盘)，在管理界面启用快捷管理后，托盘中开启的自动备份功能，是否弹出提示

## 退出到托盘
指的是你在管理游戏存档界面，点击右上角的关闭按钮时，是否退出到托盘，关闭则退出软件，不关闭则最小化到托盘，且提供托盘快捷功能

## 在应用存档时进行额外备份
覆盖存档时额外进行备份，备份位置写明在设置界面了，可以用于手动恢复

## 启用夜间模式
夜间模式现在做的不是很好，希望能有贡献者帮忙

## 在存档管理界面显示修改按钮
:::danger
不要使用该功能，这个功能在未来会被移除
:::
开启后，会在存档管理界面显示修改按钮，点击后会弹出修改存档信息的对话框，可以修改需要保存的文件，修改后可以保留以前的存档，但是只会保留和当前设置的交集部分，非常不建议使用，建议直接新建一个游戏

## 默认先删除再恢复备份（对新游戏生效）
先删除再备份值：例如你设定了一个游戏的存档是一个文件夹，在恢复存档前，会先删除这个文件夹。
而开启这个设置选项后，默认所有新存档都启用这个设定，你也可以在“存档管理-游戏名-查看受管理文件”手动更改

## 是否默认展开收藏夹
如标题含义，这个收藏夹指左侧菜单栏顶部，那个切换按钮能够切换出的收藏夹

## 生成日志文件
:::warning
注意，虽然日志文件已经做了信息脱敏，但是交给别人前请再次检查不包含个人隐私信息
:::
是否将日志写入同目录下文件，当你想要请求别人帮助你调试问题时，请务必将生成的日志一同发送。

## 调整游戏展示顺序
可以调整游戏在左栏展示的顺序
